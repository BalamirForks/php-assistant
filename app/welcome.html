Bem-vindo!

Seja bem-vindo ao PHP Assistant. Aqui estão as instruções para uso.

Esta aplicação já vem com uma versão do PHP disponível internamente.

Caso você tenha alguma outra versão, a aplicação tentará descobrir automaticamente onde o PHP está. Se não for possível, você deverá indicar onde o executável/binário do PHP está localizado.

Se você instalou ferramentas como XAMPP, pode utilizar o PHP que veio junto.

Quando estiver pronto, clique no botão "Próximo Passo".

Escolha o modo

O PHP Assistant funciona de duas maneiras: como uma aplicação comum e como uma aplicação da barra de notificações. Você pode alterar essa opção depois.

Dependendo da distribuição Linux, o modo em barra de notificações pode não funcionar.
